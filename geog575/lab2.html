Exploring Land Cover Relationships

Using Hex tiles, the tool helps explore the relationship between three land cover types in the upper midwest of the United States. Each tile is colored using a false color composite where percent of each of three land cover types is mapped to a rgb color.

  • Cultivated/Agriculutral lands are mapped to Red.
  • Wetlands are mapped to Green
  • Open Water is mapped to Blue

Rather than forcing the user to switch between different scaled maps of land cover types, I've enabled users to compare relatively subtle relationships between the land cover types.

The graph updates as you hover the mouse over each hex tile. The graph helps compare the precent of the tile covered by each habitat type and compare the value to the average value for all Hex tiles.
